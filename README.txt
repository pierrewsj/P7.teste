ATENÇÃO — Proteção de Código (Camada leve)
------------------------------------------
Você está recebendo versões minificadas dos arquivos HTML e CSS com um script simples anti-cópia/anti-inspect (bloqueio de clique direito,
atalhos comuns e detecção leve de DevTools). Isto **não** é criptografia real — é apenas para **dificultar** engenharia reversa casual.
Na Web, o código sempre é entregue ao cliente. Para camadas mais fortes, empacote como desktop (Tauri/Electron) e use ofuscadores.

Arquivos processados: index.html, interface_principal.html, controle_p7_qualidade.html, controle_p7_industrializacao.html, controle_p7_comissao_seguro.html, style.css
Coloque esta pasta 'protegido' no lugar da sua versão de produção, mantendo os mesmos nomes de arquivos.
